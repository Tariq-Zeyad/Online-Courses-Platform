NovaLearn is a complete online learning platform designed to showcase and sell educational courses. Built with a professional and modern aesthetic, it focuses on user experience, accessibility, and simplicity.

The platform is built with pure HTML, CSS, and JavaScript (Vanilla JS) without any frameworks, making it fast, lightweight, and suitable for both beginners and professionals.

## рџљЂ Getting Started

### Quick Start:
1. Download the project as ZIP or clone it
2. Extract to any folder
3. Add your photo as `Me.jpg` in the root folder (optional)
4. Open any HTML file in your browser

```bash
# Or open directly from terminal
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Using Live Server (Recommended):
```bash
# If using VS Code:
1. Install Live Server extension
2. Right-click on index.html
3. Select "Open with Live Server"
```

## рџЋЁ Customization

### Change Colors:
Open `css/styles.css` and modify the CSS variables at the top:

```css
:root {
  --primary: #7c3aed;        /* Main color */
  --primary-dark: #6d28d9;   /* Dark shade */
  --primary-light: #a78bfa;  /* Light shade */
  --accent: #06b6d4;         /* Accent color */
}
```

### Change Profile Photo:
Add your photo as `Me.jpg` in the root folder, or update the path in `about.html`.

### Update Contact Information:
Modify in all HTML files:
- Phone number
- Email address
- Social media links

### Add New Courses:
Go to `courses.html` and copy a course card template inside the `course-grid` div.
